## 体言和用言

+ 体言：名詞　数詞　代名詞
+ 用言：動詞 形容詞　形容動詞

### 形容词和形容动词

+　形容詞　＋　体言
+　形容詞の連用形：い　ー＞　く
+ 形容詞連用形　＋　動詞：美味しくて食べる
+ 形容詞連用形　＋て＋　形容動詞:  おいしくて　素敵です
+ 形容詞連用形　＋て＋　形容詞：安くて　可愛い　
+ ＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
+ 形容動詞　＋　な　＋　体言
+ 形容動詞　＋に　＋　動詞：部屋を綺麗に掃除する
+ 形容動詞　＋で　＋　形容詞：簡単でいいです
+ 形容動詞　＋で　＋　形容動詞：簡単で便利だ　複雑で不便です
